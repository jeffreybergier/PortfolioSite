Cartwheel

Open Source Carthage GUI

After building a couple of iOS apps I really wanted to learn how to build Mac apps. After seeing a presentation on Carthage from one of its creators, I knew that this would be a great project to work on. Carthage is a dependency manager for Xcode projects. The original creators built it and it only has a command line interface. I decided to tackle building a GUI front end for Carthage. Its basic, but it contains a list of Xcode projects and an update button to download and build the newest dependencies for that project.
